# Code of Conduct

## Our Pledge

We are committed to creating a respectful, professional, and welcoming environment for everyone who interacts with this repository.

## Expected Behavior

Examples of behavior that contributes to a positive environment include:

- Using respectful and professional language
- Giving constructive feedback
- Respecting different viewpoints and experiences
- Staying focused on the project scope
- Being clear, patient, and helpful in discussions

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment, intimidation, or discrimination
- Insulting, derogatory, or disrespectful comments
- Personal attacks
- Publishing private information without permission
- Sharing secrets, credentials, or confidential company information
- Spam or unrelated promotional content

## Scope

This Code of Conduct applies to:

- Issues
- Pull requests
- Discussions
- Code reviews
- Any other interaction related to this repository

## Enforcement

The repository owner is responsible for clarifying expectations and may remove, edit, or reject comments, commits, issues, or pull requests that do not align with this Code of Conduct.

The repository owner may also block contributors who repeatedly violate these expectations.

## Reporting

If you notice behavior that violates this Code of Conduct, contact the repository owner privately.

Reports should include:

- A description of what happened
- Links or screenshots where relevant
- Any additional context that may help review the issue

## Attribution

This Code of Conduct is inspired by common open-source community standards and adapted for a personal portfolio repository.
